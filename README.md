# sistem-informasi-pada-perpustakaan-digital
Tugas project web development - sistem perpustakaan

Sistem Informasi Perpustakaan Digital adalah aplikasi web yang dikembangkan untuk memudahkan pengelolaan perpustakaan secara digital. Aplikasi ini menyediakan tiga antarmuka berbeda sesuai dengan peran pengguna:

  Client - Pencarian buku, reservasi, dan riwayat peminjaman

  Staff - Kelola data buku dan transaksi peminjaman

  Admin - Manajemen pengguna dan verifikasi akun

   Fitur Utama
Client:
Dashboard personal

Katalog pencarian buku

Reservasi buku

Profil pengguna

Riwayat peminjaman

Staff:
Dashboard manajemen

Kelola data buku (CRUD)

Form tambah/edit buku

Manajemen peminjaman

Laporan transaksi

Admin:
Dashboard administrasi

Kelola pengguna

Statistik perpustakaan

Verifikasi akun pengguna

Fitur Bersama
Sistem autentikasi (Login/Register)

Layout responsif

Navigasi dinamis

Flash message notifikasi

Pagination data
